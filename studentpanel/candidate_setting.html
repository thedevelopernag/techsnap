Candidate Details

You may wish to collect some details from your Candidates before thay begin their test.Select from the list below or add a field of your own. Note these settings will show up in the candidate's report.

Add a custom field

Disclaimer

When enabled this declaration will appear on the test login page along with a checkbox.Candidates will not be able to commence this test unless they check the box, expressing assent

Test Instructions

These instruction will be displayed on the test's login page.

  1. This is a timed test.Please make sure you are not intrrupted during the test, as the timer can not be paused once started.
  2. Please ensure you have a stable internet connection .
  3. We recommend you to try the sample question for a couple of minutes before taking the main test.
  4. Before taking the test, please go through the FAQ to resolve your queries related to the test or the Hacker Rank platform.
Shuffle Questions

Each candidate will view the questions in a random order if this switch is on

Questions Points

Candidate can view the points for each questions if this switch is ON.

Copy/Paste Tracking

Enabling this will help you the pasted code in the candidate's report. We recommend to switch this on capture any cheating behaviour during the test.

Show the prompt to candidate

Enabling this will alert the candidate's if they try to copy code from other sources.

Tab Proctoring

Enabling this will track the candidte moved out of the tab during the test.
